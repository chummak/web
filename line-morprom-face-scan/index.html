คู่มือการสมัครระบบแจ้งเตือน
LINE หมอพร้อม

และแอปพลิเคชันหมอพร้อม  ·  ทำตามได้ง่าย ทีละขั้นตอน

ขั้นตอนทั้งหมด 8 ขั้นตอน

1
ลงทะเบียน eKYC และ Provider ID

ติดต่อเจ้าหน้าที่ไอทีของโรงพยาบาล เพื่อดำเนินการลงทะเบียน eKYC Health ID และ Provider ID ให้เรียบร้อย

สำคัญ: ต้องทำขั้นตอนนี้ให้เสร็จก่อน จึงจะดำเนินการขั้นต่อไปได้
2
สมัคร LINE หมอพร้อม

สมัครใช้งาน LINE หมอพร้อม และแอปพลิเคชันหมอพร้อมให้เรียบร้อย

3
เข้าสู่ระบบเว็บไซต์

เปิดเว็บเบราว์เซอร์ แล้วเข้าเว็บไซต์ด้านล่าง จากนั้นกดปุ่ม เข้าสู่ระบบ

https://scan-10917.web.app/
ต้องเชื่อมต่ออินเทอร์เน็ตผ่านเครือข่ายของโรงพยาบาลเท่านั้น
ตัวอย่างหน้าจอ
4
เลือกแผนก

เลือกแผนกของคุณที่ได้ลงทะเบียนไว้ในเครื่องสแกนใบหน้า

ตัวอย่างหน้าจอ
5
เข้าสู่ระบบด้วย Provider ID

กรอก Provider ID ที่ได้รับจากเจ้าหน้าที่ไอทีในขั้นตอนที่ 1 เพื่อเข้าสู่ระบบ

ตัวอย่างหน้าจอ
6
กรอกข้อมูลและบันทึก

เลือกแผนกและชื่อให้ตรงกับที่ลงทะเบียนไว้ในเครื่องสแกนใบหน้า กรอกข้อมูลให้ครบถ้วน แล้วกดปุ่ม บันทึกข้อมูล

ตัวอย่างหน้าจอ
7
รอการอนุมัติ

เมื่อผู้ดูแลระบบ (Admin) อนุมัติเรียบร้อยแล้ว คุณจะได้รับข้อความแจ้งเตือนทาง LINE หมอพร้อม โดยอัตโนมัติ

กรุณารอการแจ้งเตือน ไม่ต้องดำเนินการอื่นใดเพิ่มเติม
ตัวอย่างหน้าจอ
8
เริ่มใช้งานได้เลย!

เสร็จเรียบร้อย! คุณสามารถเริ่มใช้งานระบบได้ทันที

ดำเนินการสำเร็จแล้ว — พร้อมใช้งานระบบแจ้งเตือน LINE หมอพร้อม 👍
