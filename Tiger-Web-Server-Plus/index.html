1

เพิ่มพนักงานที่เครื่องสแกน

เพิ่มข้อมูลพนักงานที่เครื่องสแกน แล้วจดรหัสพนักงาน (ID) เก็บไว้เพื่อใช้ในขั้นตอนถัดไป

ตัวอย่าง: การเพิ่มพนักงานที่เครื่องสแกน
2

เข้าสู่ระบบ Tiger Web Server Plus

เปิดเว็บเบราว์เซอร์ แล้วไปที่ URL ด้านล่าง จากนั้นเข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่าน

URL http://192.168.44.8/TigerWebServerPlus/Account/Login.aspx
🔗 เปิดหน้าเข้าสู่ระบบ
ตัวอย่าง: หน้าเข้าสู่ระบบ
3

เปิดเมนูข้อมูลพนักงาน

เมื่อเข้าสู่ระบบแล้ว ให้คลิกที่เมนู "ข้อมูลพนักงาน"

ตัวอย่าง: เมนูข้อมูลพนักงาน
4

แก้ไขและบันทึกข้อมูลพนักงาน

ดำเนินการตามลำดับขั้นตอนต่อไปนี้:

ค้นหารหัสพนักงาน → กดแก้ไข → กรอกข้อมูลให้ครบ → ตั้งค่าเป็น "ผู้ใช้งาน" → บันทึก
ตัวอย่าง: การแก้ไขและบันทึกข้อมูลพนักงาน
⚠️ หมายเหตุสำคัญ ต้องเพิ่มพนักงานที่เครื่องสแกนก่อนเสมอ (ขั้นตอนที่ 1) หากค้นหาไม่พบข้อมูลพนักงาน ให้ตรวจสอบว่ารหัสพนักงาน (ID) ถูกต้องและตรงกับที่บันทึกไว้
